## 🎬 GSAP Animations Used

### Animation Types:
1. **Timeline Animations** - Sequential reveals
2. **Scroll Triggers** - Elements animate on scroll
3. **Stagger Effects** - Items appear one after another
4. **Scale Animations** - Elements grow/shrink
5. **Fade & Slide** - Opacity + Y position changes
6. **Rotation Effects** - 3D rotateX/rotateY
7. **Line Animations** - ScaleX/ScaleY for lines
8. **Floating Elements** - Random continuous motion
9. **Hover Interactions** - Scale, translate, color changes

### Easing Functions:
- `power3.out` - Smooth deceleration
- `power2.out` - Medium deceleration
- `back.out(1.4)` - Bounce effect
- `expo.inOut` - Exponential ease
- `sine.inOut` - Sine wave motion
